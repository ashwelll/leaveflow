LeaveFlow HRMS - Leave Management System
🚀 Project Overview

LeaveFlow HRMS is a full-stack web application designed to manage employee leave requests efficiently. It allows employees to apply for leave and enables HR managers to approve or reject requests in real-time. 
The system helps streamline HR workflows and improves transparency in leave tracking.

🎯 Features
👨‍💼 Employee Features:
User Registration & Login
Apply for Leave
View Leave History
Track Leave Status (Pending / Approved / Rejected)
Profile Management

🧑‍💻 HR Manager Features:
View all employee leave requests
Approve or Reject leave applications
View employee details
Monitor leave statistics
